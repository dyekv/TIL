# HTTPとは

WebサーバとWebクライアントの間でデータの送受信を行うためのプロトコル（お約束）

クライアント（ブラウザ）からサーバに送るHTTPはリクエストHTTP

サーバからクライアントに返すHTTPはレスポンスHTTP
